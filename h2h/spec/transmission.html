Length
 2 Longitud total del mensaje, incluyendo la longitud del header. Esta información viaja en binario, el orden de los bytes es de izquierda a derecha; es decir el byte de la izquierda es el más significativo y el byte de la derecha el menos significativo
Total message length, including header length. This information travels in binary, the order of the bytes is left to right; ie the left byte is the most significant and the rightmost byte the least significant

 2 Actualmente solo se ocupa el primer byte y hace referencia al checksum, si el primer byte trae un 1 indica que el checksum está activado y que esperamos un valor en el campo de checksum, si trae un 0 indica que no vamos a esperar un valor en el campo de checksum.
Currently only the first byte is occupied and refers to the checksum, if the first byte brings a 1 indicates that the checksum is activated and that we expect a value in the checksum field, if it brings a 0 indicates that we will not expect a value in the checksum field.

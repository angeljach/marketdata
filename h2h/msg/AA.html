El Filler del último registro dependerá del tamaño que conforme la última parte del comentario, la sección de control (29 caracteres) y el Indicador de la última trama (1 carácter).

The number of FEED's it depends of the registry number in the load file; that is say by valid filter.
The number of registries /sequence) for the commentary configuration, it depends on the number of characters that are in the load file.
The Filler of the last registry will depends on the size that shapes the last part of the commentary, the control section (29 characteres) and the indicator of the last trama (1 character).

El tercer valor ("e") del campo "Clasificación - Tipo Crédito Bursátil" aplica solo cuando se presentan los valores "1" o "3" del campo "Tipo de Crédito". / The first two values ("1" and "2") of the field "Classification Type - Marketability Credit" just apply when the value “Market Ability" ("2") appears in the field called "Credit Type". The third value ("e") in the field "Classification Type - Marketability Credit" just applies when the values "1" or "e" in the field "Credit Type".

[2]: El campo "Folio" agrupa la información de un mismo Filtro. / The field "Folio" groups the information of the same filter.

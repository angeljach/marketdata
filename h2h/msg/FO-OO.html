Este formato será enviado de los Sistemas de MexDer vía SETRIB a los sistemas de los socios liquidadores para confirmar la recepción y la ejecución de la confirmación de Cuenta y Posición de un hecho. El mensaje contendrá el folio del hecho al que se le asignó cuenta y posición a través del formato FN. Tendrá un código de respuesta (campo “código”), el cual tendrá el valor de cero si la transacción fue exitosa, si tuvo algún error y no se realizó la operación, el código será diferente de cero y la descripción del error estará en el campo “Mensaje”. La descripción de los diferentes códigos de error es la siguiente:

This format will be sent to the settling partner systems from MEXDER through SETRIB in order to confirm position trade reception and execution. The message will contain trade folio for which the accoun and position is allocated through the FN format. Same will have a response code ("Code" field) containing a zero for successful transaction. If there is an error, and transaction not carried out, code will be different and error description will be the "Message" field. Different error codes are as follows:

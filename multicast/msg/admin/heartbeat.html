Multicast: Mensaje Heartbeat / Heartbeat Message

Mensaje Heartbeat / Heartbeat Message

Indican la última secuencia que se ha enviado en ese grupo y puerto multicast. Estos mensajes solo se difundirán en caso de inactividad dentro del grupo y se enviarán cada cinco segundos.

El mensaje de Heartbeat solo contiene el encabezado con el campo Número de mensajes igual a cero y con la secuencia del último mensaje que se ha enviado.

The Heartbeat message will be sent every five seconds and only be disseminated in case of group inactivity. This message indicates the last sent sequence in a particular multicast group and port.

The same contains only a header with: Number of messages equal to zero and the last sent message sequence.

Multicast: Bloque de Mensaje / Message Block

Bloque / Block

El primer campo de un bloque de mensajes es la longitud del mensaje. El primer bloque de mensaje comenzará inmediatamente después del encabezado. Subsecuentes bloques de mensajes comenzaran después del último byte del bloque de mensaje previo.

First field of a message block is the message length. The first message block will start immediately after the header. Subsequent message blocks will start after the last byte of the previous message block.

Campo Offset Tamaño Valor Descripción
Longitud del mensaje
Message length
 * 2 Int16 Indica la longitud del mensaje contenido en este bloque de mensaje
Indicates message length contained in this message block
Mensajes
Total Messages
 * * El contenido del mensaje con longitud variable
Message content with variable length
